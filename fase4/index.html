Propósito del Sitio


El presente trabajo, tiene como propósito impulsar el aprendizaje de HTML5 y CSS, mediante el desarrollo de un sitio web, logrando un efecto visual agradable y armonía entre los distintos estilos que lo integran, utilizando técnicas de codificación HTML y aplicando estilos CSS, asesoradas por el Tutor. La temaría en consulta será: Etiquetas básicas de HTML, Atributos y Valores.
